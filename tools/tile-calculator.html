How to Use This Calculator

  1. Enter the length and width of the area to tile in feet.
  2. Enter your tile size in inches (assumes square tiles).
  3. Enter the grout width in inches (⅛" is standard, use 0.125).
  4. Set a waste factor — 10% for straight lay, 15% for diagonal patterns.
  5. Click "Calculate Tiles" for your total count.

The Formula

Effective Tile Size = Tile Size + Grout Width
Tiles Along = ⌈Area Length (in) ÷ Effective Size⌉
Tiles Across = ⌈Area Width (in) ÷ Effective Size⌉
Total = Tiles Along × Across × (1 + Waste%)

The calculator accounts for grout lines by adding the grout width to each tile dimension. Results are always rounded up since partial tiles still require a full tile to cut.
