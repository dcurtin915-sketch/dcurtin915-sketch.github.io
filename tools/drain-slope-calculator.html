How to Use This Calculator

  1. Select the pipe diameter for your drain line.
  2. Enter the pipe run length in feet (horizontal distance).
  3. Optionally enter a custom slope or leave blank to use the code minimum.
  4. Click "Calculate Slope" to see the required drop and compliance status.
  5. Use "Find Slope" mode to check if an existing installation meets code.

The Formula

Total Drop (inches) = Slope (in/ft) × Pipe Length (ft)
Grade (%) = (Slope in/ft ÷ 12) × 100

IPC/UPC Code Minimum Slopes

  • Pipes 2½" and smaller: ¼ inch per foot (0.25 in/ft = 2.08%)
  • Pipes 3" to 6": ⅛ inch per foot (0.125 in/ft = 1.04%)
  • Pipes 8" and larger: 1/16 inch per foot (0.0625 in/ft = 0.52%)
